# DefaultErrorFallback

The DefaultErrorFallback component provides a user-friendly error screen when unhandled exceptions occur in the application.

## Overview

The DefaultErrorFallback component is a fallback UI for React Error Boundaries. It displays a clean, branded error page that:

- Provides context about the error to the user
- Offers guidance based on user role (developer vs. end-user)
- Allows copying error details for debugging
- Maintains branding with the Graphweaver logo

## Usage

The DefaultErrorFallback component is designed to be used with React Error Boundary:

```jsx
import { ErrorBoundary } from 'react-error-boundary';
import { DefaultErrorFallback } from '@exogee/graphweaver-admin-ui-components';

const MyComponent = () => {
	return (
		<ErrorBoundary FallbackComponent={DefaultErrorFallback}>
			{/* Your component content */}
		</ErrorBoundary>
	);
};
```
