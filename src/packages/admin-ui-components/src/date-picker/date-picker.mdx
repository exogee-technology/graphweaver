# DatePicker

The DatePicker component allows users to select dates and date ranges with optional time selection.

## Overview

The DatePicker component is a versatile form control that can be used to:

- Select a single date
- Select a date range with start and end dates
- Include time selection when needed
- Accept dates in various formats
- Support internationalization with locale settings
